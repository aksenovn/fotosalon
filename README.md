# Цифровой фотоархив (Фотосалон)

Веб-приложение для автоматизации работы фотоателье. Проект включает в себя интерфейсы для клиентов (оформление и оплата заказов) и для сотрудников/администраторов (управление очередью заказов, ведение базы клиентов, каталога услуг и аналитика).

## Структура проекта

*   **`index.html`** — стартовый файл, выполняющий автоматическое перенаправление на страницу авторизации.
*   **`regandaut.html`** — страница входа в систему (авторизация) и навигационный центр.
*   **`klient.html`** — личный кабинет клиента. Позволяет выбирать фотографии, настраивать параметры печати (формат, цветность, количество), рассчитывать стоимость и симулировать оплату заказа.
*   **`sotrudnik.html`** — панель управления сотрудника (АРМ + Аналитика). Содержит базу клиентов, канбан-доску заказов, редактор каталога услуг и дашборд со статистикой для директора.

## Как запустить локально

1.  Скачайте все файлы в одну папку.
2.  Откройте файл `index.html` или `regandaut.html` в любом современном браузере.

На странице авторизации (`regandaut.html`) вы можете использовать следующие учетные данные для тестирования интерфейсов:

## Хостинг на GitHub Pages

Чтобы сайт работал в интернете:
1. Создайте публичный репозиторий на GitHub.
2. Загрузите туда все файлы проекта (включая `index.html`).
3. Перейдите в **Settings** (Настройки) репозитория -> **Pages**.
4. В разделе **Build and deployment** выберите источник `Deploy from a branch`, выберите ветку `main` (или `master`) и нажмите **Save**.
5. Через 1-2 минуты ваш сайт будет доступен по предоставленной ссылке.
